الجزء الأول: كيف يعمل الإنترنت والويب؟

قبل أن نبدأ بكتابة الأكواد، يجب أن نفهم ما يحدث خلف الكواليس عندما تفتح موقعاً على الإنترنت.

1.1

ما هو الويب؟

الويب (World Wide Web) هو نظام ضخم من الصفحات والمواقع المترابطة التي تُخزَّن على خوادم (Servers) حول العالم، ويمكن الوصول إليها عبر الإنترنت باستخدام المتصفح (Browser).

1.2

رحلة الطلب والاستجابة

الخطوةما يحدث
1 - الطلبتكتب عنوان الموقع في المتصفح (URL)
2 - DNSيُحوَّل الاسم إلى عنوان IP للخادم
3 - الطلبيرسل المتصفح طلباً (HTTP Request) للخادم
4 - الاستجابةيُرسل الخادم ملفات HTML وCSS وJS
5 - العرضيقرأ المتصفح الملفات ويعرض الصفحة
1.3

الفرق بين HTML وCSS وJavaScript

التقنيةمهمتها
HTML 🏗️هيكل الصفحة — المحتوى والعناصر
CSS 🎨تنسيق الصفحة — الألوان والأشكال والتصميم
JavaScript ⚙️التفاعل والحركة والديناميكية

مثال بسيط: المنزل = HTML (الجدران والأبواب) + CSS (الطلاء والديكور) + JavaScript (الأبواب التي تُفتح بالزر)

الجزء الثالث: إنشاء أول مشروع HTML

3.1

إنشاء مجلد المشروع

من الأفضل دائماً تنظيم مشاريعك في مجلد مخصص:

  • أنشئ مجلداً جديداً على سطح المكتب باسم: my-projects
  • داخله أنشئ مجلداً باسم: session01
  • افتح VS Code
  • اذهب إلى: File → Open Folder → اختر مجلد session01
3.2

إنشاء أول ملف HTML

في VS Code، في الشريط الجانبي:

  • اضغط على أيقونة الملف الجديد (New File)
  • سمّ الملف: index.html
  • اضغط Enter
💡 لماذا index.html ؟
اسم index.html هو الاسم الافتراضي الذي يبحث عنه المتصفح تلقائياً عند فتح أي مجلد.
عند فتح: http://example.com/ → سيبحث المتصفح تلقائياً عن index.html
3.3

هيكل صفحة HTML الأساسي

اكتب في الملف الرمز ! ثم اضغط Tab — سيُنشئ VS Code الهيكل تلقائياً (Emmet Abbreviation):

HTML 
<!DOCTYPE html>
<html lang="ar" dir="rtl">

  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>صفحتي الأولى</title>
  </head>

  <body>
    <!-- المحتوى يُكتب هنا -->
  </body>

</html>
👁 المعاينة الحيّة
3.4

شرح كل سطر بالتفصيل

الكودالشرح
<!DOCTYPE html>يُخبر المتصفح أن الملف من نوع HTML5
<html lang="ar">العنصر الجذري الذي يحتوي كل شيء | lang=ar للغة العربية
dir="rtl"اتجاه الكتابة من اليمين لليسار
<head>قسم المعلومات الخفية عن الصفحة
charset="UTF-8"ترميز النصوص - يدعم العربية والإنجليزية
viewportيجعل الصفحة متجاوبة مع الشاشات المختلفة
<title>عنوان الصفحة الذي يظهر في تبويب المتصفح
<body>قسم المحتوى الذي يراه الزائر
<!-- -->تعليق — لا يظهر في المتصفح، يُستخدم للشرح

الجزء الرابع: العناصر الأساسية في HTML

4.1

عناصر العناوين (Headings)

يوفر HTML ستة مستويات من العناوين من h1 إلى h6، حيث h1 هو الأكبر والأهم:

HTML 
<h1>عنوان رئيسي - الأكبر</h1>
<h2>عنوان ثانوي</h2>
<h3>عنوان من المستوى الثالث</h3>
<h4>عنوان من المستوى الرابع</h4>
<h5>عنوان من المستوى الخامس</h5>
<h6>عنوان سادس - الأصغر</h6>
👁 المعاينة الحيّة
📌 قاعدة مهمة

استخدم h1 مرة واحدة فقط في كل صفحة — هو العنوان الرئيسي.

الترتيب الهرمي مهم لمحركات البحث (SEO) وإمكانية الوصول (Accessibility).

4.2

عنصر الفقرة (Paragraph)

HTML 
<p>هذه فقرة نصية عادية.</p>

<p>
  يمكن أن تحتوي الفقرة على نص طويل.
  المتصفح يُنسّق المسافات والأسطر تلقائياً.
</p>
👁 المعاينة الحيّة
4.3

تنسيق النص الأساسي

الوسمالوظيفة
<strong>نص غامق مهم (Bold) — مهم لـ SEO
<em>نص مائل (Italic) — للتأكيد
<u>نص مسطّر (Underline)
<br>سطر جديد (Line Break) — وسم فارغ
<hr>خط أفقي فاصل (Horizontal Rule)
<mark>تمييز النص بلون (Highlight)
<small>نص صغير
<del>نص محذوف (يظهر بخط عليه)
HTML 
<p>
  هذا <strong>نص مهم وغامق</strong> وهذا
  <em>نص مائل للتأكيد</em> وهذا
  <mark>نص مميز بلون أصفر</mark>.
</p>

<p>السعر القديم: <del>500 ريال</del> السعر الجديد: <strong>350 ريال</strong></p>
👁 المعاينة الحيّة
4.4

الروابط (Links)

عنصر الرابط <a> هو من أهم عناصر HTML — هو ما يربط الصفحات ببعضها:

HTML 
<!-- رابط خارجي يُفتح في نافذة جديدة -->
<a href="https://www.google.com" target="_blank">اذهب إلى جوجل</a>

<!-- رابط لصفحة داخلية -->
<a href="about.html">صفحة من نحن</a>

<!-- رابط للانتقال لقسم في نفس الصفحة -->
<a href="#section1">انتقل إلى القسم الأول</a>

<!-- رابط بريد إلكتروني -->
<a href="mailto:info@example.com">راسلنا</a>

<!-- رابط هاتف -->
<a href="tel:+966501234567">اتصل بنا</a>
👁 المعاينة الحيّة
الخاصيةالشرح
hrefعنوان الرابط (URL أو مسار الملف)
target="_blank"فتح الرابط في نافذة أو تبويب جديد
target="_self"فتح في نفس النافذة (الإعداد الافتراضي)
4.5

الصور (Images)

عنصر الصورة <img> هو وسم فارغ (لا يحتاج وسم إغلاق):

HTML 
<!-- صورة من الإنترنت -->
<img src="https://example.com/photo.jpg" alt="وصف الصورة" />

<!-- صورة من نفس المجلد -->
<img src="logo.png" alt="شعار الشركة" width="200" height="100" />

<!-- صورة مع رابط -->
<a href="https://example.com">
  <img src="banner.jpg" alt="إعلان" />
</a>
👁 المعاينة الحيّة
⚠️ الخاصية alt مهمة جداً

خاصية alt تصف الصورة نصياً — تظهر إذا فشل تحميل الصورة.

محركات البحث تقرأ alt لفهم محتوى الصور.

ضرورية لإمكانية الوصول للمكفوفين الذين يستخدمون قارئات الشاشة.

الجزء الثامن: تمارين الجلسة

✏️ تمرين 1 — تدريب داخل الجلسة15 دقيقة

أنشئ صفحة HTML تحتوي على:

  • عنوان رئيسي h1 باسم مدينتك المفضلة
  • فقرة نصية تصف المدينة باختصار
  • صورة للمدينة من الإنترنت مع نص alt مناسب
  • قائمة غير مرتبة بثلاثة أماكن مشهورة فيها
  • رابط لصفحة ويكيبيديا الخاصة بالمدينة

✏️ تمرين 2 — تمرين متوسط20 دقيقة

أنشئ صفحة "قائمة طعام" تحتوي على:

  • عنوان رئيسي: قائمة مطعم [اسم مطعمك]
  • قسم المقبلات: قائمة مرتبة بـ 3 أصناف
  • قسم الأطباق الرئيسية: قائمة مرتبة بـ 4 أصناف
  • قسم الحلويات: قائمة مرتبة بـ 3 أصناف
  • استخدم hr للفصل بين الأقسام
  • أضف سعراً لكل صنف باستخدام strong

✏️ تمرين 3 — تحدي30 دقيقة

أنشئ صفحة سيرة ذاتية CV بسيطة تحتوي على:

  • اسمك كعنوان h1
  • مسمى وظيفي كعنوان h2
  • ملخص شخصي كفقرة
  • قسم المهارات: قائمة غير مرتبة
  • قسم التعليم: قائمة مرتبة من الأحدث للأقدم
  • قسم التواصل: روابط LinkedIn و GitHub والبريد الإلكتروني والهاتف
  • صورة شخصية (يمكن استخدام placeholder)

الواجب: موقع نادي ألعاب / هواية

أنشئ موقعاً مكوّناً من صفحة HTML واحدة كاملة عن هواية تحبها أو نادٍ رياضي تتابعه، يحتوي على:

◆ المحتوى الإلزامي:

  • عنوان h1 رئيسي وعناوين h2 للأقسام الفرعية

  • صورة واحدة على الأقل مع alt مناسب

  • قائمة مرتبة (ol) بخطوات أو إنجازات
  • رابطان على الأقل (داخلي وخارجي)
  • استخدام فاصل hr لتقسيم الصفحة
  • تذييل footer بسيط بحقوق النشر

◆ نقاط الإضافة (اختياري):

  • قائمة متداخلة (nested list)
  • استخدام جميع وسوم التنسيق النصي (strong, em, mark, del)
  • رابط mailto وtel
