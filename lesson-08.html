2.1

منهجية BEM لتسمية الـ Classes

BEM (Block Element Modifier) هي منهجية تسمية classes بشكل منظم تُسهّل الفهم والصيانة في المشاريع الكبيرة:

CSS 
/* BEM: Block__Element--Modifier */

/* Block — المكوّن الرئيسي */
.card { }

/* Element — جزء من Block — يُفصل بـ __ */
.card__title { }
.card__image { }
.card__body  { }
.card__footer { }
.card__btn   { }

/* Modifier — حالة مختلفة — يُفصل بـ -- */
.card--featured  { }   /* بطاقة مميزة */
.card--dark      { }   /* بطاقة داكنة */
.card--disabled  { }   /* بطاقة معطلة */

/* مثال كامل */
.service-card { }
.service-card__icon { }
.service-card__title { }
.service-card__desc { }
.service-card__price { }
.service-card--popular { border: 2px solid var(--accent); }

/* في HTML */
/* <article class="service-card service-card--popular"> */
/*   <div class="service-card__icon">🌐</div>          */
/*   <h3 class="service-card__title">تطوير الويب</h3> */
/* </article>                                          */

Custom Scrollbar (شريط التمرير المخصص)

CSS 
/* تخصيص شريط التمرير — يعمل على Chrome وEdge */

/* المسار الكامل */
::-webkit-scrollbar {
  width: 10px;           /* عرض الشريط العمودي */
  height: 10px;          /* ارتفاع الشريط الأفقي */
}

/* الخلفية (المسار) */
::-webkit-scrollbar-track {
  background: var(--color-bg);
  border-radius: 10px;
}

/* المقبض المتحرك */
::-webkit-scrollbar-thumb {
  background: linear-gradient(var(--color-secondary), var(--color-primary));
  border-radius: 10px;
  border: 2px solid var(--color-bg);
}

::-webkit-scrollbar-thumb:hover {
  background: var(--color-accent);
}

/* للمتصفحات الحديثة (Firefox) */
* {
  scrollbar-width: thin;
  scrollbar-color: var(--color-secondary) var(--color-bg);
}

Selection Styles (تنسيق النص المحدد)

CSS 
/* تنسيق النص عند تظليله بالماوس */
::selection {
  background-color: var(--color-accent);
  color: var(--color-primary);
}

/* لـ Firefox */
::-moz-selection {
  background-color: var(--color-accent);
  color: var(--color-primary);
}

/* تنسيق مختلف للعناوين */
h1::selection, h2::selection {
  background-color: var(--color-primary);
  color: white;
}

Print CSS (تنسيق الطباعة)

CSS 
/* CSS خاص بالطباعة — يُطبَّق عند طباعة الصفحة */
@media print {
  /* إخفاء العناصر غير الضرورية */
  .navbar,
  .hamburger,
  .hero-btns,
  .footer-links,
  .cta-section,
  .cookie-banner {
    display: none !important;
  }

  /* ضبط الألوان للطباعة */
  body {
    background: white;
    color: black;
    font-size: 12pt;
  }

  /* إظهار الروابط كاملة */
  a[href]::after {
    content: ' (' attr(href) ')';
    font-size: 10pt;
    color: #555;
  }

  /* تجنب قطع العناصر بين الصفحات */
  .card, article, section {
    page-break-inside: avoid;
  }

  /* هوامش الطباعة */
  @page {
    margin: 2cm;
  }
}

Smooth Scroll والـ Scroll Behavior

CSS 
/* Smooth Scroll — تمرير ناعم عند النقر على الروابط الداخلية */
html {
  scroll-behavior: smooth;
}

/* تعويض ارتفاع الـ navbar الثابت عند الانتقال للأقسام */
[id] {
  scroll-margin-top: 80px; /* = ارتفاع الـ navbar */
}

/* Scroll Snap — تثبيت التمرير عند كل قسم */
.scroll-container {
  height: 100vh;
  overflow-y: scroll;
  scroll-snap-type: y mandatory;
}

.scroll-section {
  height: 100vh;
  scroll-snap-align: start;
}

/* احترام تفضيلات التمرير */
@media (prefers-reduced-motion: reduce) {
  html { scroll-behavior: auto; }
}

  hamburger.classList.toggle('navbar__hamburger--open');

/* إغلاق القائمة عند النقر على رابط */
navLinks.querySelectorAll('.navbar__link').forEach(link => {
  link.addEventListener('click', () => {
    navLinks.classList.remove('navbar__links--open');
    hamburger.setAttribute('aria-expanded', false);
    hamburger.classList.remove('navbar__hamburger--open');
  });
});

/* ══ 2. Navbar تغيير عند التمرير ══ */
const navbar = document.getElementById('navbar');

window.addEventListener('scroll', () => {
  if (window.scrollY > 50) {
    navbar.classList.add('navbar--scrolled');
  } else {
    navbar.classList.remove('navbar--scrolled');
  }
});

/* ══ 3. Animate on Scroll — Intersection Observer ══ */
const animateElements = document.querySelectorAll('[data-animate]');

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('is-visible');
      observer.unobserve(entry.target); /* يوقف المراقبة بعد الظهور */
    }
  });
}, { threshold: 0.15 });

animateElements.forEach(el => observer.observe(el));

/* ══ 4. Active Navbar Link ══ */
const currentPage = window.location.pathname.split('/').pop() || 'index.html';
document.querySelectorAll('.navbar__link').forEach(link => {
  const href = link.getAttribute('href');
  if (href === currentPage || (currentPage === '' && href === 'index.html')) {
    link.classList.add('navbar__link--active');
  }

الجزء السادس: مراجعة شاملة لـ HTML & CSS

6.1

أسئلة المراجعة السريعة

أجب على هذه الأسئلة من الذاكرة — إذا لم تعرف إجابة فارجع للجلسة المعنية:

◆ HTML (جلسات 1-2)

السؤالالإجابة الصحيحة
ما الفرق بين <strong> و<b>؟<strong> دلالي ومهم لـ SEO | <b> تنسيقي فقط
متى تستخدم <section> ومتى <div>؟<section> لمحتوى له عنوان وسياق | <div> للتقسيم البصري فقط
ما الفرق بين method=get وpost؟get: البيانات في URL | post: مخفية (للكلمات السرية والبيانات الحساسة)
لماذا alt في الصور ضروري؟قارئات الشاشة، عند فشل التحميل، لـ SEO
ما الفرق بين <article> و<section>؟<article> مستقل يمكن نشره وحده | <section> جزء من الصفحة

◆ CSS الأساسيات (جلسة 3)

السؤالالإجابة الصحيحة
ما أولوية: class مقابل ID؟ID (100 نقطة) > Class (10 نقاط)
ما الفرق بين display:none وvisibility:hidden؟none: يحذف المساحة | hidden: يخفي لكن يبقي المساحة
ما box-sizing: border-box؟يجعل padding وborder ضمن العرض المحدد — لا يزيد عليه
ما الفرق بين margin وpadding؟margin: خارج الحدود | padding: داخل الحدود
متى نستخدم !important؟نادراً جداً — عند الاستثناء الإجباري فقط

◆ Flexbox وGrid (جلسات 4-5)

السؤالالإجابة الصحيحة
متى Flexbox ومتى Grid؟Flex: صف أو عمود واحد | Grid: صفوف وأعمدة معاً
ما justify-content: space-between؟توزيع العناصر بمسافات متساوية بينها (لا مسافة عند الأطراف)
ما الوصفة الذهبية لشبكة متجاوبة؟repeat(auto-fit, minmax(280px, 1fr))
ما flex: 1؟flex: 1 1 0% — العنصر ينمو وينكمش بحرية
كيف تجعل عنصراً يمتد كل أعمدة Grid؟grid-column: 1 / -1

◆ Responsive وAnimations (جلسات 6-7)

السؤالالإجابة الصحيحة
ما Mobile-First؟نكتب CSS الجوال أولاً ونستخدم min-width لإضافة التعديلات
ما الفرق بين rem وem؟rem: نسبة للجذر (html) | em: نسبة للأب المباشر
ما clamp(min, preferred, max)؟قيمة مرنة بحدود دنيا وعليا — تُغني عن Media Queries
ما الخصائص الآمنة للتحريك؟transform وopacity فقط — لأنهما يستخدمان GPU
ما animation-fill-mode: forwards؟العنصر يبقى عند آخر حالة في الـ keyframes بعد انتهاء الأنيميشن
