
# Front-End Master Book — مسار HTML &amp; CSS

موقع تفاعلي لتعليم HTML وCSS باللغة العربية، بديل عن عروض PowerPoint — إعداد **م. حسن حجار**.

هذا مسار **منفصل** عن موقع JavaScript (نفس الهوية البصرية والمؤلف، لكن ملفات مستقلة تماماً — لا تدمج مجلدَي الموقعين معاً عند الرفع، أبقهما في مجلدين/مستودعين منفصلين).

## التشغيل

افتح `index.html` مباشرة في أي متصفح حديث. لأفضل تجربة مع المعاينة الحيّة (iframe)، شغّل خادماً محلياً بسيطاً بدل الفتح المباشر من القرص:

```bash
npx serve .
# أو
python3 -m http.server 8000
```

## الفرق الجوهري عن موقع JavaScript

- **زر "جرّب الكود"** في موقع JavaScript ينفّذ الكود ويعرض ناتج `console.log`.
- هنا بدّلته بزر **"👁 معاينة حيّة"**: يعرض الكود مُصيَّراً فعلياً داخل `<iframe>` — مناسب لطبيعة HTML/CSS البصرية.
- عندما يظهر مثال HTML متبوعاً مباشرة بمثال CSS في نفس الفقرة، تُدمَج المعاينة تلقائياً (الكودان معاً في معاينة واحدة).
- الأمثلة القائمة بذاتها (HTML كامل، أو مقتطف CSS بلا HTML مرافق) تُعرض بزر معاينة مستقل أو بدون معاينة إن تعذّر تصوّر السياق.
